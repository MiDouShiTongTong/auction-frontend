## 可用脚本

- 安装所需依赖包

  ```shell
  npm i
  ```

- 安装热部署工具

  ```shell
  npm i nodemon
  ```

- 运行开发环境

  ```shell
  npm run start
  ```

- 运行生产环境

  ```shell
  npm run build
  ```
